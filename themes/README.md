# Themes & Icons

Classic ArcoLinux look targets:

- GTK theme: Arc-Dark / Arc
- Icons: Sardi (Sardi-Arc, Sardi-Qogir, etc.)
- Cursor: Bibata-Modern-Ice or similar

## Sources (secured locally)

- arc-theme-master.zip  in WD secured-zips/themes-icons/
- Sardi-master.zip      in WD secured-zips/themes-icons/

## Quick install ideas

From official repos when available:

paru -S arc-gtk-theme bibata-cursor-theme

Sardi often comes from Nemesis / AUR / local packages.

Keep wallpapers and custom icon packs in ../assets/ if needed.
