# ArcoLegacy Scripts

Helper scripts adapted from the classic ArcoLinux Nemesis collection.

Intended for Arch-based systems (especially ARCOTIX / Arconet / Artix).

## Current Scripts

| Script | Description | Usage |
|--------|-------------|-------|
| fix-ssh-permissions.sh | Fixes common SSH directory and key permissions | ./fix-ssh-permissions.sh |
| give-me-pacman.conf.sh | Generates a clean, minimal pacman.conf | ./give-me-pacman.conf.sh |
| install-yay-or-paru.sh | Installs an AUR helper (yay or paru) | ./install-yay-or-paru.sh |
| get-me-started.sh | General helper for new systems | ./get-me-started.sh |
| diagnose-boot.sh | Helps diagnose boot-related issues | ./diagnose-boot.sh |
| check-boot-config.sh | Checks current boot configuration | ./check-boot-config.sh |

## How to Use

Make the script executable, then run it:

chmod +x script-name.sh
./script-name.sh

Or copy them to ~/bin and run from anywhere.
